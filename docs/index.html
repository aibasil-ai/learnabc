系統用途

本系統提供孩子進行 A-Z 英文字母學習，並透過「發音練習 → 挑戰驗證 → 獎勵影片」建立穩定學習循環。家長可透過 PIN 保護的管理介面調整學習節奏與獎勵條件，讓孩子在可控環境下持續練習。

教學重點

以下重點對應實際操作順序，避免跳步造成學習中斷。

先聽再學會

先播放字母與單字發音，再按「學習完成」進入挑戰題。

進度即時可查

進度、星星、解鎖狀態與字母地圖可隨時切換查看。

家長設定集中管理

可調整 PIN、影片秒數、解鎖門檻、影片連結與播放方向。

3 步驟操作流程

步驟 1

進入學習主畫面

先播放字母與單字發音，建立該字母最基本記憶，再進入下一步。

步驟 2

完成挑戰驗證

按下學習完成後回答挑戰題，答對才會計入學習並自動跳下一個字母。

家長調整參數

在家長管理面板調整獎勵影片與 PIN，維持學習動機與規範邊界。

常見問題

沒有聲音怎麼辦？

先確認裝置音量，並先點擊一次畫面。部分瀏覽器需互動後才允許語音播放。

PIN 無法通過？

若未修改過，預設 PIN 為 1234；若已修改，請以最新 PIN 為準。

影片無法播放？

請確認網路狀態、YouTube 連結格式，並確保影片允許嵌入。
