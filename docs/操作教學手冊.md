# ABC 冒險樂園 操作教學手冊

本手冊說明孩子學習流程、獎勵影片機制與家長設定方式。

> 截圖環境（已重新拍攝）：  
> 手機版 `390 x 844`、桌機版 `1366 x 900`

## 1. 啟動方式

1. 啟動專案：

```bash
npm run dev
```

2. 開啟瀏覽器：`http://127.0.0.1:8000`

<img src="./manual-images/01-學習主畫面.png" alt="學習主畫面" width="260" />

## 2. 預設設定

- 幾個字母解鎖一次影片：`3`
- 每次影片秒數：`30`
- 預設獎勵影片：`https://youtu.be/-yG4mBzGwq8?si=8MhQ4I5Wby-D-3wm`
- 家長 PIN：`1234`

## 3. 孩子學習流程

### 3.1 學習主畫面

- 顯示字母、對應單字與圖片。
- 必須先按「字母發音」與「單字發音」，才能按「學習完成」。
- 若該字母已學過，右上角會顯示「已學習過」。

<img src="./manual-images/01-學習主畫面.png" alt="學習主畫面" width="260" />

### 3.2 學習進度

- 可查看學習數量、分數、星星與獎勵解鎖狀態。
- 解鎖後可按「立即播放獎勵影片」。

<img src="./manual-images/02-學習進度.png" alt="學習進度" width="260" />

### 3.3 字母地圖

- 點任一字母可快速切換到該字母。
- 已學過字母會有完成狀態。

<img src="./manual-images/03-字母地圖.png" alt="字母地圖" width="260" />

### 3.4 小測驗

- 題目會語音朗讀。
- 選項作答後可看到結果提示。

<img src="./manual-images/04-小測驗.png" alt="小測驗" width="260" />

## 4. 學會挑戰（學習完成）

### 4.1 開啟挑戰

1. 先播放字母發音與單字發音。
2. 按「學習完成」後，跳出挑戰題目。

<img src="./manual-images/05-學會挑戰.png" alt="學會挑戰" width="260" />

### 4.2 答錯

- 顯示答錯提示文字。
- 會播放錯誤音效。

<img src="./manual-images/06-學會挑戰答錯.png" alt="學會挑戰答錯" width="260" />

### 4.3 答對

- 會播放答對音效。
- 該字母才算完成，並自動跳到下一個未學過字母。

## 5. 家長管理介面

### 5.1 進入家長專區

1. 點右上角家長圖示（🛡️）。
2. 輸入 PIN（預設 `1234`）。

### 5.2 PIN 錯誤提示

- PIN 錯誤時，輸入框下方會顯示錯誤訊息。

<img src="./manual-images/08-PIN錯誤提示.png" alt="PIN 錯誤提示" width="260" />

### 5.3 管理項目

- 每幾個字母解鎖一次影片
- 每次影片秒數
- YouTube 影片連結 / 影片 ID
- 影片方向（橫向 / 直向）
- 變更 PIN
- 啟用或停用影片獎勵

## 7. 常見問題

### 7.1 沒有聲音

- 先確認裝置音量。
- 某些裝置需要先點擊畫面一次，才會允許語音或音效播放。

### 7.2 PIN 無法通過

- 先確認 PIN 是否已被修改。
- 未修改時預設 PIN 為 `1234`。

### 7.3 影片無法播放

- 檢查網路是否正常。
- 檢查 YouTube 連結格式是否正確。
- 部分影片不允許嵌入播放，請改用其他影片。
