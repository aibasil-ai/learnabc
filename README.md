# ABC 冒險樂園

適合 6 歲以下兒童的英文字母學習網頁遊戲。

## 主要功能

- 26 個英文字母 + 對應開頭單字與圖片
- 字母發音、單字發音、題目朗讀（中英混合）
- 學會挑戰機制（答對才算完成該字母）
- 學習進度、分數、星星獎勵
- YouTube 獎勵影片（可續播、可鎖定播放控制）
- 家長 PIN 管理介面（門檻、秒數、影片、方向、啟用與重設）

## 開發啟動

```bash
npm run dev
```

開啟：`http://127.0.0.1:8000`

## 測試

```bash
npm test
```
